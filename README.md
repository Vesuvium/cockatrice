# Medusa

Elixir static site generator that supports Pug.

## Installation

```
git clone https://github.com/Vesuvium/medusa
cd medusa
mix deps.get
mix escript.install
```

## Usage

Start with:

```
medusa new mysite
```

A compile command is available:

```
medusa compile
```

A dev server is provided:

```
medusa server
```
